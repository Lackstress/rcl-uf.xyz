## üöÄ Quick Start

### Local Development (Windows)

Double-click `local.bat` or run:

```bash
npm install
npm run dev
```

Then open http://localhost:3000

### Local Development (Linux/Mac)

```bash
chmod +x local.sh
./local.sh
```

## üì¶ Deployment

### VM Deployment (Linux)

1. Upload the project to your VM
2. Make the deploy script executable:
   ```bash
   chmod +x deploy.sh
   ```
3. Run the deployment:
   ```bash
   ./deploy.sh
   ```

The script will:
- Install Node.js if needed
- Install dependencies
- Build the project
- Configure Nginx
- Set up the site at rcl-uf.xyz

### Manual Build

```bash
npm install
npm run build
```

The built files will be in the `dist` folder.
